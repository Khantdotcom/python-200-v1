# Python 200: Learning Objectives

## Course Objectives

By the end of this course, you will be able to:

1. Perform exploratory data analysis and statistical hypothesis testing on real-world datasets using pandas, NumPy, and Matplotlib.
2. Build, evaluate, and tune supervised machine learning models (regression and classification) using scikit-learn.
3. Save trained models to disk and load them in a separate prediction workflow, separating training from deployment.
4. Interact with large language model (LLM) APIs, build chatbots with conversation memory, and apply prompt engineering techniques.
5. Extend LLMs with retrieval-augmented generation (RAG) so they can answer questions grounded in documents outside their training data.
6. Build tool-using AI agents that reason, call external functions, and complete multi-step tasks autonomously.
7. Explain the cloud computing service model (IaaS/PaaS/SaaS) and navigate a major cloud provider portal.
8. Connect Python applications to a cloud-hosted relational database and perform secure, idempotent reads and writes.
9. Integrate ML models and LLM APIs as stages in a batch data pipeline applied to cloud-stored records.
10. Orchestrate a complete Extract–Transform–Load (ETL) pipeline with Prefect, adding retries, structured logging, and production-readiness patterns.
